# jms-ci

> JMS 发布工具（可持续集成）

### 开始使用

安装:

```bash
npm install -g jms-ci
```

启动：

确保本地端口 `11234` 没有被占用，否则应用将无法启动！

```bash
jms-ci
```

成功启动后，本地默认浏览器会自动开启 `http://localhost:11234` 。

**如默认浏览器版本低于IE9，则不能正常使用！建议使用现代浏览器如：chrome、360浏览器等。**
